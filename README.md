# CDS_rules
Here are  rules developed for a vaccination clinical decision support system during the EUVABECO pilot projects.

They are provided for information about how such rules can be expressed, using a custom [domain specific language](grammar.md) developed during the project.

Please note that these rules are owned jointly by SYADEM and by the implementing partner:
  + Riga Stradins University for Latvia
  + FRATEM for Belgium
  + University of Thessaly for Greece
  + Jagellonian University for Poland
  
Any reuse would require the prior approval of these owners.
